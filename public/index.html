Promotion Type

There are three types of promotion and one option to not use a code (if you wish to only use the communication cell feature and control group feature). The promotion types are "Online only", "Instore only" and "Online & Instore". You will be taken through a setup process to associate a promotion to your email template.

Select the type of promotion
Email Template
Select the template that you intend to use this voucher on. Only Email templates that contain "BAU" in there name will appear here. If your email template does not appear here please rename it.
Control Group Data Extension Name
Select the Data Extension that will log Control Group users. The data extension must reside in "Data Extensions > BAU Journey Data > BAU Control Groups".
